# Lumière Dorée V2

Refonte React + Vite + Tailwind CSS + Supabase.

## Démarrage

```bash
npm install
npm run dev
```

Copier `.env.example` vers `.env.local` et y mettre les clés Supabase (étapes suivantes).

## Structure

- `src/` — application React
- `legacy/` — prototype HTML/CSS validé par le client (référence visuelle)

## Branches (cahier des charges)

Travailler sur `feature/ui` ou `feature/content`, jamais sur `main` directement.
